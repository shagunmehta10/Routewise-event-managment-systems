## 🎭 Animation Sequences

### Shimmer Effect (2s loop)
```
Frame 1:  ░░░░░░░░░░░░░░░░
Frame 2:  ▒░░░░░░░░░░░░░░░  ← Light sweeps right
Frame 3:  ░▒░░░░░░░░░░░░░░
Frame 4:  ░░▒░░░░░░░░░░░░░
Frame 5:  ░░░▒░░░░░░░░░░░░
...       → → → → → → → → →
Frame 16: ░░░░░░░░░░░░░░▒░
Repeat from Frame 1
```

### Float Animation (3s loop)
```
Position Y:
  0s: ▲ (0px)     ← Start
  0.75s: │         ← Going up
  1.5s: ▼ (-10px)  ← Peak
  2.25s: │         ← Coming down
  3s: ▲ (0px)     ← Back to start
  Repeat...
```

### Glow Pulse (2s loop)
```
Shadow Intensity:
  0s:   ●○○○○ (20px blur)  ← Dim
  0.5s: ●●○○○ (26px blur)
  1s:   ●●●●● (40px blur)  ← Bright
  1.5s: ●●○○○ (26px blur)
  2s:   ●○○○○ (20px blur)  ← Dim again
  Repeat...
```

### Gradient Shift (3s loop)
```

  0s:   [████▓▓▒▒░░]  0% 50%

  3s:   [████▓▓▒▒░░]  0% 50%    ← Back
```

## 🌟 Icon Style Guide

```
Size:        20px (small) | 24px (medium) | 32px (large)
Stroke:      2px
Color:       Match section theme
Hover:       Scale(1.1) + Color shift
Animation:   Rotate | Float | Pulse

Example:
  Normal:  📍 (teal, 24px)
  Hover:   📍 (teal-light, 26px, rotated 5°)
```
