## 🆘 Troubleshooting

### Port Already in Use
```bash
npx kill-port 5173
npm run dev
```

### Module Not Found
```bash
rm -rf node_modules package-lock.json
npm install
```

### TypeScript Errors
1. Press `Ctrl+Shift+P` in VS Code
2. Type: "TypeScript: Restart TS Server"
3. Press Enter
